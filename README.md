# Genrate-baby-names

This project was done to help my famliy to pick new names to their newborn



Input: Requires you to input lettes

  Input list = v --> vowls. 
             = c --> constants.
             = l --> any other letter.
             = UPPER CASE LETTERS WILL BYPASS PROGRAM.


Output: Genrate 6 diffent names from given input
